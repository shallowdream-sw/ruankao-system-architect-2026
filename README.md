# 系统架构设计师备考包

本仓库用于执行 2026 下半年「系统架构设计师」备考计划。默认从 2026-07-04 开始，到 2026-10-23 完成冲刺；考试窗口按软考官网公布的 2026-10-24 至 2026-10-27 处理，具体日期以准考证为准。

官方考试安排参考：[2026 年度计算机技术与软件专业技术资格（水平）考试工作计划](https://www.ruankao.org.cn/article/content/2603051204118156470700001.html)。

## 当前结论

- 当前应该报名的考试：**系统架构设计师**。
- 目标场次：**2026 年下半年**。
- 当前职业背景：数据开发，工作内容与 SQL、Flink、数据链路、调度系统、数据平台稳定性和性能优化相关。
- 学习节奏：每周 10-12 小时。
- 当前项目只聚焦考试，不展开软著、专利或杭州 E 类成果材料。

## 固定周节奏

按每周 10-12 小时设计：

- 工作日：每天 1 小时，适合看知识点、做选择题、整理错题。
- 周六：3 小时，适合案例题或专题深挖。
- 周日：3 小时，适合复盘、论文、整套模拟。

建议每周日晚上完成一次周复盘：记录本周投入小时数、做题正确率、薄弱专题、下周最重要的 3 件事。
