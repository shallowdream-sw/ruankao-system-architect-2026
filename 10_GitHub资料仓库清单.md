# GitHub 资料仓库清单

更新时间：2026-07-04。

用途：整理系统架构设计师、软考高级相关的公开 GitHub 仓库，作为备考资料补充。

说明：这些仓库不是官方来源，内容可能包含考生回忆版、个人笔记、机构资料索引或外链。使用时以“辅助理解和刷题”为主，正式考试安排、报名、教材信息仍以软考官网和省级软考办通知为准。

## 优先级 A：先看这些

### 1. xxlllq/system_architect

链接：https://github.com/xxlllq/system_architect

检索时信息：Stars 8565；Forks 2072；最近更新时间 2026-07-03；描述为 2026 年系统架构设计师（软考高级）备考资料。

适合用途：找系统架构设计师备考全套资料索引；查历年真题和解析目录；看论文、讲义、备考心得、机考操作指南等资料分类。

注意：仓库体量较大，可能包含视频、教材、外链和大文件。7 月阶段不要全量下载、全量看，优先看目录、历年真题、讲义和备考心得。

## 7 月使用建议

本阶段只建议这样用 GitHub 资料：

1. 从 `xxlllq/system_architect` 或 `xiaomabenten/system_architect` 找一份目录完整的系统架构设计师资料。
2. 找综合知识真题或刷题入口，每天做 10-20 道选择题。
3. 找一份基础讲义，对照《系统架构设计师教程（第2版）》补概念。
4. 不看冲刺押题，不背论文范文，不同时打开 5 个仓库。

推荐组合：

- 主资料仓：`xxlllq/system_architect` 或 `xiaomabenten/system_architect` 二选一。
- 刷题仓：`ruankaodaren/ruankao`。
- 笔记补充：`lisahust/FDExam_Note` 或 `hakusai22/System_Architect`。

## 风险提醒

- GitHub 仓库内容不等于官方资料。
- 真题多为回忆版，答案解析不一定是官方标准答案。
- 大仓库可能包含大量二进制文件，不建议直接全量 clone。
- 涉及教材、视频、讲义等资料时，注意版权和来源。
